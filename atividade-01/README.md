# Atividade 01 - API Calculadora em Python

Esta atividade consiste em uma API desenvolvida em Python com Flask que recebe dois números e uma operação matemática, retornando um JSON com o resultado.

## Funcionalidades

A API realiza as seguintes operações:

- soma
- subtracao
- multiplicacao
- divisao

## Arquivos da atividade

- `app.py` → código principal da API
- `requirements.txt` → dependências do projeto
- `exemplos.json` → exemplos de requisições
- `README.md` → documentação da atividade
