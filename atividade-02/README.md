# Atividade 02

Análise de dados de vendas de uma livraria com Python, Pandas e Matplotlib.

## Conteúdo desenvolvido

- Criação e exploração de dataset
- Análise de faturamento, categorias, vendedores e regiões
- Visualizações gráficas
- Desafios extras:
  - evolução do faturamento mês a mês
  - gráfico de tendência de vendas
  - vendedor com maior ticket médio
  - análise de vendas com total acima de 200
  - inserção e tratamento de valores nulos

## Como executar

```bash
python analise.py
